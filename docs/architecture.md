# Архитектурные ограничения
## Gateway
Stateless, не хранит данные и не знает об их расположении

## Metadata Serivce 
Хранит только метаданные и не участвует в передаче содержимого объектов

## Storage Node 
Отвечает исключительно за хранение и выдачу объектов 

## Взаимодействие
Все сервисы внутри взаимодействуют по gRPC, Gateway предоставляет HTTP S3-like API
