# 02 - Metadata Service
Решил добавить больше эндпоинтов S3, а именно:
- `GET /` - получить список бакетов
- `PUT /{bucket}` - создать бакет
- `GET /{bucket}` - получить список объектов бакета
- `HEAD /{bucket}` - получить метаданные бакета
- `HEAD /{bucket}/{key...}` - получить метаданные объекта
